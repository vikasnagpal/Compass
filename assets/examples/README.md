# Example screenshots for the Vibe Curator impact page

The four-wins slideshows in `vibe-curator-impact.html` pull real screenshots from
this folder. **Save your image files here with these exact names** (`.jpg`).
Until a file exists, the page shows an on-brand fallback tile with the filename
on it — so nothing ever looks broken.

| Filename | Slideshow | What to put here |
|---|---|---|
| `calendar-1.jpg` | Calendar | Screenshot of the **Daily Activities** section on a Zostel listing (e.g. Pokhara — Ludo Legends / Sarangkot Sunrise / River Dancing) |
| `calendar-2.jpg` | Calendar | Another **Daily Activities** screenshot (e.g. Kotgarh — Zostel Gully Bash / Fooseball Face-Off / Bonfire) |
| `activity-1.jpg` | Activities | Photo of an art & journaling circle |
| `activity-2.jpg` | Activities | Photo of a game / poker night |
| `activity-3.jpg` | Activities | Group photo (e.g. the beach day out) |
| `social-1.jpg` | Social | Instagram profile screenshot — `zo.nepal` |
| `social-2.jpg` | Social | Instagram profile screenshot — `zostel_kotgarh` |

Notes
- The **Reviews** slideshow uses real Google reviews transcribed directly into the
  page (Amrit Singh, Pavitra Kulkarni, Ayush Singh) — no image files needed there.
- Tall screenshots (calendar, social) are anchored to the **top** so the most
  important part stays visible; activity photos are centre-cropped.
- Keep file sizes reasonable (ideally < 400 KB each) so the page stays fast.
- If you prefer a different name/extension, update the matching `<img src="...">`
  in `vibe-curator-impact.html` (search for `assets/examples/`).
